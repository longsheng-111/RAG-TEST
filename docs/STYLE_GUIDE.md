## 第一章 · 设计原则（全文灵魂）

### 1.1 审美立场：个人项目的"小而美"

这不是一个要讨好所有人的产品，是一本**某个人自己的作业本**。三条立身之本：

- **手工感**：界面应该像手做出来的，不像渲染出来的。允许贴纸歪 2 度、印章盖下去有点斜、圆角偶尔不对称；不允许任何地方看起来像"模板默认"。
- **克制而确定的细节**：细节必须少而准。一个页面有一个记忆点（一枚印章、一条像素墨水条）就够；堆十个装饰等于没有装饰。每个细节必须能说出"它为什么在这里"。
- **有世界观的趣味**：趣味来自"作业本冒险"这一套自洽隐喻（见第二章），不来自随机表情包、流行梗或浮夸动效。

### 1.2 明确反对："大众网页感"（反 AI 感立场）

以下形态在本项目中视为**事故**，视同 bug 处理：

- **模板化布局**：千篇一律的卡片栅格 + 渐变 hero + 圆角阴影卡片，去掉 logo 后可以是任何产品的界面。
- **默认控件形态**：antd 默认蓝（`#1677ff`）、默认深灰 Tooltip 气泡、浏览器原生 `alert()`、未改造的 Radio/Spin/分页器——它们一出现，作业本的幻觉立刻破产（现状证据见体检报告 1.5、1.6）。
- **无叙事的中性界面**：文案只有"确定/取消/操作成功"，空态只有"暂无数据"，等待只有转圈——功能正确但没有任何世界观，这是"AI 生成的后台"的典型气味。

### 1.3 每条原则的自检方法

新增或修改任何组件时，逐条自问：

| 原则 | 判断方法 |
|------|---------|
| 手工感 | **眯眼测试**：眯起眼看这个页面，有没有一个元素是"活"的（贴纸、印章、像素数字）？如果满眼都是规整矩形，不够"作业本"。 |
| 克制而确定的细节 | **减法测试**：把这个装饰删掉，页面是否依然成立？成立才准留；说不清理由的装饰一律删。 |
| 有世界观的趣味 | **映射表测试**：这个元素/文案能对应到第二章映射表的哪一行？对不上表的"趣味"是杂音。 |
| 反默认件 | **默认件排查**：这个界面里有没有任何一个控件是"拿去就能用的 antd 默认样子"？有即不通过。 |
| 反中性叙事 | **朗读测试**：把页面所有文案朗读一遍，像不像一个"作业本里的讲解员"在说话？像企业后台公告即不通过。 |

## 第九章 · 已裁决记录

> 本章原为四项待定事项，2026-07-17 由项目所有者全部拍板定稿，结论已移入正文（第二/三/四/六章），文档升 v1.1。本章存档备查；日后推翻任一裁决按第七章纪律升大版本。

### 裁决-1 · 引用角标 `rotate(±2deg)`（原待定-1，报告 1.7）

- **决策**：保留 ±2deg 倾斜（采原写法 A）。引用角标正式归入贴纸类，第二章映射表登记为"线索贴纸"，沿用 hover 回正规范；贴纸/印章可倾斜，**正文容器永不倾斜**。落实处：第二章映射表与贴纸类定义边界、4.2。
- **用户理由（精简）**：手工感最强，角标与 persona 贴纸气质统一；以"正文容器永不倾斜"把边界划死，可读性代价可接受。
- **裁决日期**：2026-07-17

### 裁决-2 · `--ink-faint` 对比度修复（原待定-2，报告 4.2）

- **决策**：加深为 `#8A7A63`（采原写法 A），对纸底约 4.6:1，达 WCAG AA 4.5:1；适用场景定为弱提示正文与禁用文字。代码现状 `globals.css:16` 仍为 `#A3937A`，随体检报告 B6 整改。落实处：3.1。
- **用户理由（精简）**：一处改动全局达标、语义不变；接受与 `--ink-secondary`（约 5.4:1）层次差距缩小的代价。
- **裁决日期**：2026-07-17

### 裁决-3 · 边框宽度口径（原待定-3，报告 1.10）

- **决策**：1.5px 为全站唯一口径（采原写法 A），分隔线 1px 为唯一例外，2px 特例全部收编。现状 2px 出现处为待收编项：`layout.tsx:108` 下划线、印章描边 `QAPanel.tsx:1879`、引用块左边条 `globals.css:508`。落实处：3.3、4.3。
- **用户理由（精简）**：口径最简、最好审；接受印章/标题下划线等"重笔"场景失去更粗表现力的代价。
- **裁决日期**：2026-07-17

### 裁决-4 · 知识库标签非对称圆角 `3px 4px 2px 5px`（原待定-4，`KnowledgeBaseManager.tsx:303`）

- **决策**：保留为"手剪贴纸"特许特例（采原写法 A），登记为唯一特许的非对称圆角元素；**仅限贴纸类标签，禁止扩散到其他组件**。`.op-tag` 漂移副本仍随 B3 清除。落实处：3.3、4.2。
- **用户理由（精简）**：手工感记忆点 +1；以"特许特例 + 禁止扩散"成文，防仿效扩散。
- **裁决日期**：2026-07-17
